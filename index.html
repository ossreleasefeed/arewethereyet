arewethereyet

simple, zero-dependency progress meter

What?

AWTY is a simple, small (~716 bytes minified), Youtube style, progress element you can quickly and easily add to your own projects.

There are zero dependencies, just add the JS, a little bit (~10 lines) off CSS, and you are good to go. Click the buttons to see it in action.

Tested to work on IE8+ and the rest of the world.

How?

Add the needed JS file and CSS snippet and then intialize it like this:


var options = {
    selector: '.top',
    bgcolor: red
};
awty.add(options);

Now everytime your task progresses, simply call the update function, incrementing the value.


awty.update(percentage);
